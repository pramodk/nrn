3 # NEURON
4 NEURON is a simulator for models of neurons and networks of neuron. See [http://neuron.yale.edu](http://neuron.yale.edu) for installers, source code, documentation, tutorials, announcements of
5 courses and conferences, and a discussion forum.
6 
7 ## Installing NEURON
8 
9 NEURON provides binary installers for Linux, Mac and Windows platforms. You can find the latest
10 installers for Mac and Windows [here](https://neuron.yale.edu/ftp/neuron/versions/alpha/). For
11 Linux and Mac you can install the official Python 3 wheel with:
12 
13 ```
14 pip3 install neuron
15 ```
16 
17 If you want to build the latest version from source, currently we are supporting two build systems:
18 
19 - CMake (__recommended__)
20 - Autotools (legacy, minimum support - __will be DROPPED in the next release__)
21 
22 Note that starting with the 8.0 release, CMake is used as the primary build system for NEURON.
23 We would be grateful for any feedback or issues you encounter using the CMake-based build system.
24 Please [report any issue here](https://github.com/neuronsimulator/nrn/issues) and we will be
25 happy to help.
26 
27 **If you are using autotools, we strongly recommend switching to CMake as of now.**
28 
29 For detailed installation instructions see [docs/install/install_instructions.md](docs/install/install_instructions.md) file.
